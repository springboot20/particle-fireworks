# Canvas Particle Fireworks
Canvas Fireworks is an interactive animation project built with the HTML5 Canvas API and vanilla JavaScript. Clicking or tapping anywhere on the canvas generates colorful firework explosions composed of animated particles that radiate outward, gradually shrink, and fade away. The project includes responsive high-DPI canvas rendering, particle physics with friction, synchronized sound effects, and smooth animations optimized for both desktop and mobile devices.

## ✨ Features

- 🎆 Interactive firework particle explosions
- 🎨 Randomized colors, sizes, and velocities
- 📱 Responsive canvas with high-DPI support
- 👆 Mouse and touch (pointer) interaction
- 🔊 Integrated firework sound effects
- ⚡ Smooth animations using `requestAnimationFrame`
- 🧹 Automatic particle cleanup for better performance
- 🛠️ Built with HTML5 Canvas and Vanilla JavaScript
